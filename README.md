# MSIS Portfolio Website

A professional portfolio website built with Django and Bootstrap 5.

## Features
- Home page with hero section and expertise overview.
- About page with detailed education and experience.
- Projects grid managed via Django Admin.
- Resume page with embedded PDF viewer.
- Contact form with email notifications and database logging.
- Integrated Chatbase AI chatbot.

## Setup Instructions
1. Clone the repository.
2. Create a virtual environment: `python -m venv venv`.
3. Activate the environment: `.\venv\Scripts\activate` (Windows) or `source venv/bin/activate` (Mac/Linux).
4. Install dependencies: `pip install -r requirements.txt`.
5. Run migrations: `python manage.py migrate`.
6. Create a superuser: `python manage.py createsuperuser`.
7. Start the server: `python manage.py runserver`.
